# QGIS 실내 도면 제작 가이드

제1공학관(ENG1) GeoJSON을 QGIS에서 만드는 step-by-step 가이드.

**핵심 원칙: QGIS에서는 지오메트리(도형)만 정확하게 그린다. 속성(방 번호, 타입 등)은 Python 스크립트로 일괄 입력한다.**

---

## 1. QGIS 설치

1. https://qgis.org → **QGIS LTR (Long Term Release)** 다운로드/설치
2. 설치 후 실행

---

## 2. Google Satellite 타일 추가

1. 왼쪽 **탐색기** 패널 → **XYZ Tiles** 우클릭 → **새 연결**
2. 설정:
   - 이름: `Google Map`
   - URL: `https://mt1.google.com/vt/lyrs=m&x={x}&y={y}&z={z}`
   - 최고 확대/축소 수준: `21`
   - 확인
3. **Google Map**을 더블클릭해서 레이어에 추가
5. 제1공학관 위치로 이동: 하단 **"위치를 찾으려면 입력하십시오"** 바에 `14134827.7, 4479925.0` 입력, 축척 1:5000

---

## 3. 구조도 이미지 지오레퍼런싱

구조도 JPG를 위성사진 위에 정확히 겹쳐놓는 과정.

1. 메뉴 → **레이어** → **지리참조자...** (Georeferencer 창이 별도로 열림)
2. **Open Raster** → `SKKU_building_structure diagram_resize/제1공학관_1.jpg` 선택
3. **Add Point** 도구 선택
4. 구조도에서 건물 모서리 클릭 → **From Map Canvas** → 위성사진에서 같은 모서리 클릭
5. **최소 4개 점** 찍기 (건물 각 꼭짓점):
   - 21동 남서쪽 모서리
   - 21동 남동쪽 모서리
   - 23동 북서쪽 모서리
   - 23동 북동쪽 모서리
6. **Transformation Settings** (톱니바퀴 아이콘):
   - Transformation type: **Polynomial 1** (4점이면 충분)
   - Target CRS: **EPSG:3857** (Web Mercator)
   - Output: 적당한 경로에 GeoTIFF로 저장
   - ✅ Load in project when done
7. **Start Georeferencing** (재생 버튼) 클릭
8. 지오레퍼런싱된 이미지가 위성사진 위에 겹쳐서 표시됨
9. 레이어 패널에서 투명도 조절: 레이어 우클릭 → Properties → Transparency → 50-70%

---

## 4. 스냅 설정 (핵심!)

정확한 벽 연결을 위한 스냅 설정.

1. 먼저 스냅 툴바 표시: **보기(V)** → **툴바** → **스냅 툴바** 체크
2. 상단에 나타난 **자석 아이콘(🧲) 클릭**하여 스냅 활성화
3. 설정:
   - 스냅 모드: **모든 레이어**
   - 스냅 대상: **꼭짓점** + **선분** 둘 다 체크
   - 허용 오차: **12 픽셀**
4. **위상 편집** 버튼 활성화 — 인접 폴리곤의 공유 꼭짓점이 같이 움직여서 틈이 안 생김
5. **고급 디지타이즈 작업** 패널 켜기: **보기** → **패널** → **고급 디지타이즈 작업** 체크
6. 패널 맨 위 아이콘 줄의 **첫 번째 아이콘 클릭**하여 활성화
7. 톱니바퀴(⚙️) 아이콘 → **스냅 작업 우선 순위** → **15.0, 30.0, 45.0, 60.0°** 선택 — 이전 선분 기준 15/30/45/60/90도에 자동 스냅

## 5. 레이어 생성

**속성 필드는 최소한으로** — 나중에 스크립트로 채운다.

### 5.1 건물 외곽선 레이어
1. **레이어** → **레이어 생성** → **새 Shapefile 레이어**
2. 설정:
   - File name: `eng1_outline.shp`
   - Geometry type: **Polygon**
   - CRS: **EPSG:4326 (WGS 84)**
3. 속성 필드: **추가 안 해도 됨** (스크립트에서 자동 입력)
4. OK

### 5.2 방(Rooms) 레이어
- File name: `eng1_rooms_L1.shp`
- Geometry type: **Polygon**
- CRS: **EPSG:4326**
- 속성 필드: **없음** (스크립트에서 자동 입력)

### 5.3 복도(Corridors) 레이어
- `eng1_corridors_L1.shp`, Polygon, EPSG:4326
- 속성 필드: **없음**

### 5.4 벽(Walls) 레이어
- `eng1_walls_L1.shp`
- Geometry type: **LineString**
- CRS: **EPSG:4326**
- 속성 필드: **없음**

### ~~5.5 POI 레이어~~ → 사이트 에디터에서 작업
POI(계단, 엘리베이터, 문, 화장실)와 네비게이션 그래프는 사이트 에디터에서 직접 배치한다.

## 5.5 프로젝트 저장 (여기까지 설정 저장)

여기까지 설정이 끝나면 프로젝트를 저장해두면 다음에 바로 트레이싱부터 시작할 수 있다.

1. **프로젝트** → **다른 이름으로 저장** → `cad/eng1.qgz`
2. 다음 작업 시: `eng1.qgz` 더블클릭하면 모든 설정 복원 (타일, 구조도, 스냅, 레이어)

## 6. 트레이싱 (그리기만!)

**QGIS에서는 도형만 정확하게 그린다. 속성은 입력하지 않는다.**

### 6.1 건물 외곽선
1. `eng1_outline` 레이어 선택 → **편집 모드 켜고끄기** (연필 아이콘 ✏️)
2. **폴리곤 피처 추가** 선택
3. 위성사진/구조도에서 건물 외곽을 따라 클릭 (모서리마다)
4. 다 그리면 **우클릭**으로 완료 (잘못 찍었으면 Backspace)
5. 속성 팝업이 뜨면 그냥 OK (비워두기)
6. **레이어 편집 내용 저장** (💾)

### 6.2 방 (3점 직사각형 + 피처 분할)

#### 준비
1. **보기(V)** → **툴바** → **형태 디지타이징 툴바** 체크
2. **설정(S)** → **키보드 단축키...** → "분할" 검색 → **피처 분할**에 단축키 지정 (예: `X`)

#### Step 1: 3점 직사각형으로 큰 영역 그리기
1. `eng1_rooms_L1` 레이어 선택 → **편집 모드 켜고끄기**
2. 형태 디지타이징 툴바 → **직사각형 아이콘 옆 ▼** → **3점에서 직사각형 추가**
3. 첫 점: 외곽선(또는 기존 방) 꼭짓점에 스냅
4. 둘째 점: 외곽선을 따라 다른 꼭짓점에 스냅 (한 변 방향 결정)
5. 셋째 점: 높이 방향으로 드래그 → 클릭
6. 속성 팝업 → 그냥 OK

#### 직사각형 모드 해제
- 디지타이징 툴바의 **▼ 드롭다운** → **선분으로 디지타이즈** 선택하면 일반 폴리곤 모드로 복귀

#### Step 2: 피처 분할로 방 나누기
1. **피처 분할** 선택 (단축키 또는 **편집** → **도형 편집** → **피처 분할**)
2. 윗변 선분에 스냅해서 첫 점 클릭
3. **`A`** → **`90`** → 🔒 잠금 — 윗변 기준 직각 유지
4. 아랫변을 관통하도록 클릭 → 우클릭 완료
5. 반복해서 방 개수만큼 분할
6. **레이어 편집 내용 저장** (💾)

#### 계단 영역
- 계단도 `eng1_rooms_L1`에 폴리곤으로 그린다 (별도 레이어 X)
- 속성 입력 시 `room_type: "stairs"`로 구분

### 6.3 복도
- 같은 방법으로 복도 영역을 폴리곤으로 그리기

### 6.4 벽
- `eng1_walls_L1` 레이어 → **라인 피처 추가**
- 벽을 따라 LineString 그리기

### ~~6.5 POI~~ → 사이트 에디터에서 배치
계단/엘리베이터/문/화장실은 사이트 에디터에서 지도 위에 직접 클릭하여 배치한다.

## 7. 유용한 팁

### 고급 디지타이즈
1. **보기** → **패널** → **고급 디지타이즈 작업** 체크
2. 패널 맨 위 아이콘 줄의 **첫 번째 아이콘 클릭**하여 활성화
3. **편집 모드 + 피처 추가 + 첫 점 클릭** 후부터 사용 가능

### 스냅 각도 설정 (핵심!)
고급 디지타이즈 패널의 톱니바퀴(⚙️) 아이콘 클릭 → **스냅 작업 우선 순위** → 각도 선택:
- **45.0, 90.0, 135.0, 180.0°** — 직각 건물에 적합
- 이 설정을 하면 선을 그릴 때 **이전 선분 기준으로** 45/90도에 자동 스냅됨
- 건물이 회전되어 있어도 이전 선분 기준이라 정확하게 직각이 유지됨

### 점 되돌리기
- **`Backspace`** — 마지막으로 찍은 점 취소 (여러 번 누르면 여러 점 취소)

### 레이어 스타일 (작업 중 구분용)
- 레이어 우클릭 → **Properties** → **Symbology**
- rooms: 파란색 반투명, corridors: 회색 반투명, walls: 검정 실선, pois: 빨간 점

## 8. GeoJSON 내보내기

### 레이어별 내보내기
1. 레이어 우클릭 → **내보내기** → **다른 이름으로 피처 저장...**
2. 설정:
   - Format: **GeoJSON**
   - CRS: **EPSG:4326**
   - File name: 예) `eng1_rooms_L1.geojson`
3. OK

**모든 레이어를 개별 GeoJSON으로 내보내기:**
- `eng1_outline.geojson`
- `eng1_rooms_L1.geojson`
- `eng1_corridors_L1.geojson`
- `eng1_walls_L1.geojson`

## 9. 속성 입력 (Claude Code)

QGIS에서 내보낸 GeoJSON에는 지오메트리만 있다. **Claude Code에서 구조도 이미지를 보고 직접 GeoJSON을 수정**한다.

### Claude Code가 하는 일:
1. **레이어 파일명 → indoor 타입 매핑** (`rooms` → `"room"`, `corridors` → `"corridor"` 등)
2. **level 입력** (파일명의 L1, L2에서)
3. **구조도 이미지를 보고 방 번호(ref) + 방 유형(room_type) 직접 입력**
4. **Feature ID 생성** (`room/eng1_L1_0` 형식)
5. **건물 외곽선 메타데이터 입력** (`building`, `name`, `loc_ref` 등)
6. **전체 레이어 합치기** → 단일 `eng1.geojson` 출력

### 사용법:
1. QGIS에서 내보낸 GeoJSON 파일들을 `cad/` 폴더에 넣기
2. Claude Code에게 요청:
   ```
   cad/ 폴더의 GeoJSON 파일들에 속성을 입력해줘.
   구조도는 SKKU_building_structure diagram_resize/제1공학관_1.jpg 참고.
   ```
3. Claude Code가 구조도 이미지를 읽고 각 폴리곤의 위치를 보고 방 번호/타입을 매칭
4. 합쳐진 `eng1.geojson` 출력

## 10. 앱에 적용

1. Claude Code가 합친 `eng1.geojson`을 앱에 복사:
   ```bash
   cp cad/eng1.geojson 2.5d_indoor_navigation_frontend/public/geojson/eng1.geojson
   ```

2. 앱에서 확인:
   ```bash
   cd ../2.5d_indoor_navigation_frontend
   node index.js
   ```
   → `http://localhost:3000` 접속

3. 확인 사항:
   - 건물 외곽선이 올바르게 표시되는가?
   - 방이 색상별로 구분되는가?
   - 방 번호가 표시되는가?
   - 3D 모드에서 벽이 렌더링되는가?

## 11. 앱 호환성 체크리스트

최종 GeoJSON이 앱에서 올바르게 렌더링되려면 (스크립트가 자동으로 채움):

| 체크 | 항목 | 값 | 누가 채우나 |
|------|------|------|------------|
| ☐ | 건물 외곽선 | `building: "university"` + 메타데이터 | 스크립트 |
| ☐ | Feature ID | `room/eng1_L1_0` 형식 | 스크립트 |
| ☐ | indoor | `room` / `corridor` / `wall` | 스크립트 (파일명 기반) |
| ☐ | level | `"1"`, 멀티레벨은 `"1;2"` | 스크립트 (파일명 기반) |
| ☐ | ref | `"21101"` 등 방 번호 | Claude Code (구조도 보고 입력) |
| ☐ | room_type | `classroom` / `lab` / `office` 등 | Claude Code (구조도 보고 입력) |
| ☐ | wall_type | `partition` / `exterior` / `corridor` | Claude Code |
| ☐ | 좌표계 | EPSG:4326 | QGIS 내보내기 |
| ☐ | Polygon 닫힘 | 첫 좌표 = 마지막 좌표 | QGIS 자동 |

## 작업 순서 요약

```
1. QGIS 설치 + Google 타일 추가          (10분)
2. 구조도 지오레퍼런싱                     (15분)
3. 스냅 설정                              (2분)
4. 레이어 5개 생성                         (5분)
5. 도형 트레이싱 (외곽선+방+복도+벽)        (30-60분)
6. GeoJSON 4개 내보내기                    (5분)
7. Claude Code로 속성 입력 + 합치기         (5분)
8. 앱에서 확인                             (1분)
─────────────────────────────────────────────
총: 1층당 약 70-100분 (첫 층), 이후 30-60분/층
```

## 층 작업 순서

1. 1층 완성 → 앱에서 검증
2. 다른 층은 1층 레이어를 복사한 뒤 달라지는 방만 수정 (대부분의 벽/복도는 동일)
3. 5개 층 모두 완성 후 전체 합치기
