## Map Interaction

- **방 클릭** → 팝업: `[📍 출발로 설정] [🏁 도착으로 설정]`
- **헤더 입력** + **지도 클릭** 모두 출발/도착 설정 가능
- **검색 placeholder**: "방 번호 입력 (ex: 21517)"

## Video Player (Apple Look Around Pattern)

### Default: 상하 분할
- 상단 50%: 360° 비디오 (드래그로 회전)
- 하단 50%: 2.5D 맵 + 경로 + 현재 위치 동기
- 확대 버튼(↗)으로 전체화면 전환

### Full-screen: 전체 화면
- 축소 버튼(↙)으로 분할 모드 복귀
- [↻ 시점 초기화] 버튼 (우측 상단)

### Controls
- seekbar: 전체 클립 통합, 클립 번호 표시 (clip 2/5)
- 기본 시점: 이동 방향(전방)
- ◀ ▶ 이전/다음 클립 버튼

---

## Route Animation

- **Duration**: 2초 polyline draw, ease-out timing
- **Multi-floor**: 현재 층 = 진한 파랑(#42A5F5), 다른 층 = 반투명(opacity 0.3)
- **Stairs**: 계단 노드에 수직 연결선
- **Floor transition**: 자동 (경로 애니메이션과 동기, 계단 도달 0.5s 전에 전환)

## Interaction States

| Feature | Loading | Empty | Error | Success |
|---------|---------|-------|-------|---------|
| 검색 자동완성 | N/A (로컬) | placeholder | "해당 방 없음" | 드롭다운 |
| 경로 표시 | 헤더 스피너 | N/A | "네트워크 오류" + 재시도 | 2s 애니메이션 |
| 360° 비디오 | sphere+스피너 | N/A | "영상 불러오기 실패" | 360° 재생 |
| 층 전환 | 0.3s fade | N/A | N/A | 새 층 렌더링 |

### Error Messages (Korean)
- 검색 없음: "해당 방을 찾을 수 없습니다. 다른 번호를 입력해보세요."
- API 오류: "네트워크 오류. 인터넷 연결을 확인하세요." + 재시도 버튼
- 비디오 실패: "영상을 불러올 수 없습니다. 지도에서 경로를 확인하세요."
